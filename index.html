📖 What is shashx?

shashx is a beginner-friendly, custom-designed programming language aimed at demystifying how programming languages work under the hood. It is built with simplicity and clarity in mind — offering clean syntax, custom logic structures, and real-time execution through an interpreter model.

Developed by Shashwat Singh, shashx serves as both a personal project and a powerful educational tool that helps aspiring developers understand the core concepts behind language design.

  • 🔤 Inspired by modern scripting languages like Python and JavaScript
  • 🧠 Designed to teach how interpreters, tokenizers, and parsers work from scratch
  • 🧪 Ideal for experimenting with syntax rules, logic flow, and runtime behaviors
  • 🎯 Great for students, hobbyists, and developers curious about building languages
  • 📚 Comes with documentation, VS Code support, and examples

Whether you're a curious developer or an enthusiast interested in compiler theory, shashx opens the doors to understanding what's behind every line of code you write — by letting you build and run your own.

✨ Features

  • ✅ Dynamically typed – no need to declare types
  • ⚡ Interpreted – executes directly without compilation
  • 🧼 Custom clean syntax – simple and intuitive commands
  • 🔁 Loop support – flp, wlp, dlp
  • 🧠 Conditionals – if, elf, el
  • 🖨️ Print statements with pf()
  • 🛠️ Built with JavaScript (Node.js)
  • 🖥️ CLI-based execution
  • 💻 VS Code syntax extension support

🔧 Under the Hood

shashx is built from scratch using JavaScript and follows a basic interpreter pattern. Here's how it works internally:

  • 📜 Lexer: Breaks your code into tokens (keywords, symbols, identifiers).
  • 🌳 Parser: Builds an Abstract Syntax Tree (AST) from tokens.
  • 🧠 Interpreter: Walks the AST and executes commands in real-time.
  • 📁 File Extension: Uses .sx for source code files.
  • 🧪 Minimal runtime: Everything runs via a lightweight runtime powered by Node.js.

This structure makes shashx simple, fast, and easy to extend!

📦 Installation

Install via NPM (Recommended)

  npm install -g shashx

Run a .sx File

  shashx hello.sx

💡 Example: hello.sx

hello.sx

  pf("Hello from shashx!");

Terminal Output

  Hello from shashx!

🧠 Syntax Guide

➕ Print Statement

pf("Hello, World!");

📦 Variable Declaration

@x = 10
@y = 20
pf(>x)
pf(>y)

➰ For Loop (flp)

flp @i = 0 to 3 {
  pf("flp i = " > i);
}

📝 Comment Line

~> This is a single-line comment

Use ~> to write single-line comments in shashx.

🔁 While Loop (wlp)

  @j = 0;
  wlp j < 2 {
    pf("wlp j = " > j);
    j = j + 1;
  }

🔁 Do-While Loop (dlp)

  @k = 0;
  dlp {
    pf("dlp k = " > k);
    k = k + 1;
  } wlp k < 2;

🔂 Conditional Statements

  @x = 10
  if x > 5
    pf("Greater than 5")
  elf x == 5
    pf("Equals 5")
  el
    pf("Less than 5")
